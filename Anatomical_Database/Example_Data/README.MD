## Example Data
Example data files for the schema (JSON format) and raw data files
- **Atlantic herring**
  - [aherr001.dat](https://github.com/nmfs-ost/AA-SI_echoSMs/blob/main/Anatomical_Database/Example_Data/aherr001.dat)
    - Original data file for an Atlantic herring (*Clupea harengus*) that was used as input to the KRM model IDL code developed my J. Jech and J. Horne.
  - [Clupea_harengus_bd.json](https://github.com/nmfs-ost/AA-SI_echoSMs/blob/main/Anatomical_Database/Example_Data/Clupea_harengus_bd.json)
    - the JSON-format file that has the required metadata as defined in the schema for the body ("bd") anatomical feature
  - [Clupea_harengus_bd.toml](https://github.com/nmfs-ost/AA-SI_echoSMs/blob/main/Anatomical_Database/Example_Data/Clupea_harengus_bd.toml)
    - the toml-format file that merges metadata, data, and WoRMS information that is used in the echoSMs KRM model of the body ("bd") anatomical feature
  - [Clupea_harengus_sb.json](https://github.com/nmfs-ost/AA-SI_echoSMs/blob/main/Anatomical_Database/Example_Data/Clupea_harengus_sb.json)
    - the JSON-format file that has the required metadata as defined in the schema for the swimbladder ("sb") anatomical feature
- **lavnun**
  - **[lavnun](https://github.com/nmfs-ost/AA-SI_echoSMs/blob/main/Anatomical_Database/Example_Data/lavn01.dat)**
    - Original data file for a lavnun (*Acanthobrama spp.*) that was used as input to the KRM model IDL code developed my J. Jech and J. Horne.
- **pilchard**
  - **[pilchard](https://github.com/nmfs-ost/AA-SI_echoSMs/blob/main/Anatomical_Database/Example_Data/plch11.dat)**
    - Original data file for a pilchard (*Sardina pilchardus*) that was used as input to the KRM model IDL code developed my J. Jech and J. Horne.
